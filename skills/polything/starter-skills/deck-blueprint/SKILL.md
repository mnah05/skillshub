# deck-blueprint

## When to use
- When you need a sales deck or pitch deck outline fast.

## Inputs
- outputs/client_brief.md (or paste the brief)

## Output
- outputs/deck_blueprint.md (use templates/deck_blueprint.md)

## Instructions
1. Create 8–12 slides (default 8).
2. For each slide: title + bullets + proof placeholder.
3. Keep it simple and story-led.
