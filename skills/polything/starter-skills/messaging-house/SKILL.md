# messaging-house

## When to use
- When messaging is drifting or you need consistent copy across channels.

## Inputs
- inputs/icp_and_segments.md
- inputs/offer_notes.md
- Optional: inputs/discovery_brief.md

## Output
- outputs/messaging_house.md (use templates/messaging_house.md)

## Instructions
1. Pick ONE segment to focus on.
2. Write a one-liner positioning statement.
3. Add 3–5 proof points (or label “proof needed”).
4. Add objections + practical reframes.
5. Add 3 CTA options.
