# experiment-designer

## When to use
- When you want to validate messaging fast without guessing.

## Inputs
- outputs/messaging_house.md
- outputs/offer_page_blueprint.md (optional)
- Constraints: budget, time, volume

## Output
- outputs/experiment_plan.md (includes a test card)

## Instructions
1. State the hypothesis (use the one-liner positioning).
2. Pick the smallest viable test (e.g., email CTA test, simple landing page, £50–£200 ad test).
3. Define metrics, pass threshold, and stop rules.
4. Output the plan + a Test Card using templates/test_card.md.
